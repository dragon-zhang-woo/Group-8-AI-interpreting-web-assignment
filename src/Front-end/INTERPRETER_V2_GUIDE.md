# AI 口译训练助手 V2 — 功能及使用文档

## 概述

`interpreter_v2.html` 是一个**纯前端**AI 口译训练单页应用（SPA），零后端依赖，双击即用。通过浏览器内置的 Web Speech API 和免费在线翻译 API，实现中英双向翻译、口译实战演练、自动评分与学习记录管理。

**版本**：V2  
**技术栈**：HTML5 + CSS3 + JavaScript (ES6+)  
**文件大小**：约 97 KB（1302 行）  
**推荐浏览器**：Google Chrome 或 Microsoft Edge（语音识别依赖）

---

## 技术架构

### 核心 API

| API | 用途 | 提供方 | 费用 |
|-----|------|--------|------|
| **Web Speech API** | 语音识别（STT） | 浏览器内置（Chrome/Edge 内核） | 免费 |
| **SpeechSynthesis API** | 语音合成（TTS） | 浏览器内置 | 免费 |
| **MyMemory API** | 文本翻译 | Translated.net | 免费（每天约 1000 次） |
| **LibreTranslate** | 翻译回退 | libretranslate.de | 免费 |

### 数据存储

| 存储方式 | 用途 | 容量 |
|----------|------|------|
| **localStorage** | 学习记录持久化 | 浏览器限制（通常 5-10MB） |
| **CSV 导出** | 记录备份与离线分析 | 无限制 |

## 三大功能模块

### 模块一：AI 翻译机

**功能描述**：中英双向翻译，支持文本输入和语音输入两种方式，翻译结果可播放语音朗读和复制。

**操作流程**：

```
┌─ 选择翻译方向 ───────────────────────────┐
│  英 → 中：英文输入，输出中文翻译           │
│  中 → 英：中文输入，输出英文翻译           │
└──────────────────────────────────────────┘
          ↓
┌─ 选择输入方式 ───────────────────────────┐
│                                           │
│  📝 文本输入：                             │
│    1. 在文本框输入/粘贴内容                │
│    2. 点击"开始翻译"                      │
│    3. 查看翻译结果                        │
│                                           │
│  🎤 语音输入：                             │
│    1. 点击"开始录音" 对麦克风说话          │
│    2. 点击"停止录音"                      │
│    3. 点击"识别语音" 转为文字              │
│    4. 点击"翻译此文本"                    │
│    5. 查看翻译结果                        │
│                                           │
└──────────────────────────────────────────┘
          ↓
┌─ 翻译结果操作 ───────────────────────────┐
│  🔊 播放翻译语音（浏览器朗读）             │
│  📋 复制翻译文本到剪贴板                   │
└──────────────────────────────────────────┘
```

**翻译回退机制**：MyMemory API → LibreTranslate API（自动切换，用户无感知）

### 模块二：实战演练

**功能描述**：核心口译训练模块。用户阅读英文原文 → 用中文口译 → 系统同步录音并实时识别 → 三维度自动评分 → 给出改进建议。

**操作流程**：

```
1️⃣ 选择练习难度
   ├── ⭐ 简单：日常问候对话
   ├── ⭐⭐ 中等：商务/科技类
   └── ⭐⭐⭐ 困难：学术/演讲类

2️⃣ 阅读英文原文
   系统从内置素材库随机抽取对应难度的英文段落

3️⃣ 录制中文翻译（录音 + 实时识别同步）
   ├── 点击"开始录音并识别"
   ├── 对着麦克风说出你的中文翻译
   │   └── 屏幕实时显示识别中间结果
   ├── 点击"停止"
   │   ├── ✅ 音频已保存（显示文件大小）
   │   └── ✅ 识别文本自动填入输入框
   └── 若识别不准，可手动修改文本

4️⃣ 提交评分
   ├── 显示参考翻译
   ├── 三维度评分展示
   └── 针对性改进建议

5️⃣ 保存记录
   记录存入 localStorage，可在"学习记录"查看
```

#### 评分系统详解

评分范围：每个维度 **0-3 分**，总分 = 三维度平均值

**发音标准性（0-3 分）**

| 条件 | 加分 |
|------|------|
| 翻译文本长度 ≥ 5 个字符 | +1 |
| 无连续 4 次以上重复同一字符（排除卡顿） | +1 |
| 包含合理标点符号（，。！？等） | +1 |

**语言流畅性（0-3 分）**

| 条件 | 加分 |
|------|------|
| 包含结束标点（表示完整句） | +1 |
| 长度与参考译文比例 ≥ 30% | +1 |
| 用字多样性 ≥ 40%（避免重复） | +1 |

**翻译准确性（0-3 分）**

| 相似度阈值 | 得分 |
|-----------|------|
| ≥ 0.8 | 3.0 分 |
| ≥ 0.6 | 2.5 分 |
| ≥ 0.4 | 2.0 分 |
| ≥ 0.2 | 1.0 分 |
| < 0.2 | 0.5 分 |

相似度算法：Jaccard 字符相似度 × 0.4 + Bigram 序列相似度 × 0.6

**综合评语等级**

| 总分范围 | 评语 |
|----------|------|
| ≥ 2.5 | 表现优秀！翻译质量较高，继续保持。 |
| ≥ 2.0 | 表现良好，部分细节有待完善。 |
| ≥ 1.5 | 表现尚可，建议多加练习以提高准确度。 |
| < 1.5 | 需要改进，建议对照参考译文仔细揣摩。 |

### 模块三：学习记录

**功能描述**：查看所有历史练习记录，包含统计面板、详情弹窗和 CSV 导出。

**统计面板**：
- 练习次数：历史总练习次数
- 平均得分：所有练习总分的平均值
- 最高得分：历史最高总分

**记录表格**：

| 列名 | 说明 |
|------|------|
| 时间 | 练习完成的日期时间 |
| 原文 | 英文原文前 28 字符预览 |
| 发音 | 发音标准性得分 /3 |
| 流畅 | 语言流畅性得分 /3 |
| 准确 | 翻译准确性得分 /3 |
| 总分 | 三维度平均分 |
| 操作 | "查看"按钮打开详情弹窗 |

**详情弹窗**包含：完整时间、英文原文、你的翻译、各维度得分、评语

**CSV 导出**：
- 文件名格式：`口译练习记录_YYYY-MM-DD.csv`
- 编码：UTF-8（含 BOM，Excel 可正确打开）
- 列：时间, 原文, 你的翻译, 发音, 流畅, 准确, 评语

## 常见问题

### Q1：点击录音按钮后提示"无法访问麦克风"
**A**：浏览器需要麦克风权限。请在浏览器弹窗中选择"允许"，或在浏览器设置中允许该网页使用麦克风。

### Q2：语音识别结果不准确
**A**：Web Speech API 的识别质量受以下因素影响：
- 环境噪音：尽量在安静环境中使用
- 发音清晰度：语速适中，咬字清晰
- 浏览器：Chrome 的识别准确度通常优于 Edge
- 可以在识别完成后手动修正文本框中的内容

### Q3：翻译结果显示英文而非中文
**A**：检查"翻译方向"是否选择了正确的方向（英→中 / 中→英）。

### Q4：MyMemory API 翻译失败
**A**：系统会自动回退到 LibreTranslate。如两者均失败，请检查网络连接或等待片刻重试（免费 API 有调用频率限制）。

### Q5：实战演练评分总是偏低
**A**：评分系统基于文本相似度算法，不是 AI 语义理解。要获得高分，建议：
- 尽量完整翻译原文含义
- 使用标点符号分隔句子
- 避免过度简化或遗漏关键信息

### Q6：学习记录会丢失吗
**A**：记录存储在浏览器的 localStorage 中。以下情况会导致记录丢失：
- 清除浏览器数据
- 使用隐私/无痕模式
- 更换浏览器或设备
- 建议定期使用"导出 CSV"功能备份

### Q7：能否在手机/平板上使用
**A**：可以。界面已适配响应式布局（≤768px 自动切换为移动端布局）。语音功能需要移动端 Chrome/Edge 浏览器支持。
