# AI 口译练习网站 - 功能文档

## 📱 应用概述

这是一个完整的 HTML5 网络应用，实现了核心口译功能，包括：
- 🌐 AI 翻译机（支持英中互译）
- 💪 实战演练（自动评分系统）
- 📊 学习记录（本地数据持久化）

**无需任何外部 API 或后端服务** - 所有功能在浏览器中本地运行！

---

## 🎯 核心功能

### 1️⃣ AI翻译机 (Tab 1)

#### 输入方式
有两种输入方法可选：

**A. 文本输入模式**
- 直接输入或粘贴要翻译的文本
- 支持英→中、中→英两个方向
- 实时翻译处理

**B. 语音输入模式**
- **实时录音**：点击"开始录音"按钮即可录制语音
  - 支持实时计时显示
  - 点击"停止录音"结束
  - 自动保存为音频 Blob
  
- **文件上传**：
  - 支持拖拽上传音频文件
  - 支持点击选择文件
  - 支持格式: MP3, WAV, OGG, M4A
  
- **语音识别**：
  - 使用 Web Speech API (Chrome/Edge/Safari 原生支持)
  - 自动将录音或上传的音频转换为文字
  - 实时显示识别进度

#### 翻译与输出
- 点击"翻译"按钮翻译识别的文字
- **双重输出**：
  - 📝 文本显示（可复制）
  - 🔊 实时语音播放（浏览器 Web Speech API）
  - 💾 音频文件下载（可将翻译结果保存为音频）

---

### 2️⃣ 实战演练 (Tab 2)

**材料库**（5 个预设主题）：
1. 日常问候 - Daily Greetings
2. 商务会议 - Business Meeting  
3. 旅游导览 - Tourism Guide
4. 技术演讲 - Tech Speech
5. 医疗咨询 - Medical Consultation

**使用流程**：
1. 从下拉菜单选择一个材料
2. 查看英文原文和中文参考翻译
3. 在文本框输入你的翻译
4. 点击"提交评分"获得智能评分

**评分系统** (9分制 × 3个维度)：
- **准确性**：翻译内容是否准确传达了原意
- **流畅性**：表达是否自然流畅，有无断句卡顿
- **发音规范**：字词是否使用得当，有无明显错误
- **总体得分**：三项平均分

**自动反馈**：系统会根据得分生成个性化的评语和改进建议

---

### 3️⃣ 学习记录 (Tab 3)

**统计面板**：
- 📊 练习次数
- 📈 平均得分
- 🏆 最高得分

**详细记录表**：
- 显示所有练习的时间、材料、各维度得分
- 点击"查看"查看完整的翻译详情和反馈
- 按时间倒序排列

**数据管理**：
- ✅ 自动保存到浏览器本地存储（LocalStorage）
- ✅ 数据在浏览器关闭后仍会保留
- ✅ 支持清除所有记录功能

### 前端技术

## 📊 数据流图

```
┌─────────────────────────────────────────────────────┐
│         AI 口译练习网站 - 数据流                      │
└─────────────────────────────────────────────────────┘

【输入层】
  ├─ 文本输入 → 直接翻译

        ↓

  ├─ 文字显示
  ├─ 实时语音播放
  ├─ 音频文件下载
  └─ 评分结果展示

        ↓

【存储层】
  └─ 本地存储（LocalStorage）
```

## 📝 示例场景

### 场景 1：日常问候翻译
1. 打开 AI 翻译机
2. 选择"中→英"
3. 输入或录制："你好，今天天气真好"
4. 获得翻译："Hello, what nice weather we have today"
5. 点击"播放 & 下载"听和保存翻译

### 场景 2：商务练习
1. 打开实战演练
2. 选择"商务会议"材料
3. 仔细阅读英文原文
4. 输入你的中文翻译
5. 提交获得 9 分制评分和反馈
6. 在学习记录中查看历史成绩

### 场景 3：语音识别练习
1. 打开 AI 翻译机
2. 切换到"语音输入"模式
3. 点击"开始录音"
4. 用英文说一句话（例如："Good morning, how are you?"）
5. 点击"停止录音"
6. 系统自动识别为文字
7. 点击"翻译"获得中文翻译
8. 播放听翻译结果
