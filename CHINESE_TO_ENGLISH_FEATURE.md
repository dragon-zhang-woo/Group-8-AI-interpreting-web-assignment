# 中译英功能使用指南

## 📋 功能概述

为 AI 口译训练助手 V2 的**实战演练**模块增加了**中→英翻译模式**，与原有的**英→中模式**并存。用户现可灵活选择翻译方向进行口译训练。

### 新增功能
- ✅ 翻译方向选择：英→中 (中文翻译) / 中→英 (英文翻译)
- ✅ 中文原文显示：当选中"中→英"时显示中文原文
- ✅ 英文语音识别：根据翻译方向自动切换语言识别（中文/英文）
- ✅ 双向评分：根据用户翻译与对应语言的参考翻译进行评分

## 🚀 快速开始

### 1. 打开应用
```bash
# Windows
start src/Front-end/interpreter_v2.html

# macOS
open src/Front-end/interpreter_v2.html

# 或双击文件直接打开
```

### 2. 进入实战演练
- 点击页面顶部的"**实战演练**"标签页

### 3. 选择翻译方向

#### 英→中模式（原有功能）
```
1. 点击 "英→中 (中文翻译)" 按钮
2. 选择练习难度（简单/中等/困难）
3. 查看英文原文
4. 点击"开始录音并识别"
5. 用中文说出翻译
6. 点击"停止"完成
7. 点击"提交评分"
```

#### 中→英模式（新增功能）
```
1. 点击 "中→英 (英文翻译)" 按钮
2. 选择练习难度（简单/中等/困难）
3. 查看中文原文
4. 点击"开始录音并识别"
5. 用英文说出翻译
6. 点击"停止"完成
7. 点击"提交评分"
```

## 🎤 语音识别提示

### 英文识别建议
- 发音清晰，语速适中（不要过快）
- 尽量在安静环境中使用
- 避免背景噪音
- 使用 Chrome 或 Edge 浏览器以获得最佳效果

### 中文识别建议
- 正常语速即可
- 尽量连贯表达
- 避免过多停顿
- Chrome 识别准确度通常优于 Edge

## 📊 评分系统

评分基于三个维度（各 0-3 分，总分为三维度平均值）：

### 发音标准性
- ✅ 翻译文本长度 ≥ 5 字符 → +1 分
- ✅ 无连续重复字符 → +1 分
- ✅ 包含合理标点符号 → +1 分

### 语言流畅性
- ✅ 包含结束标点（表示完整句） → +1 分
- ✅ 长度与参考翻译比例 ≥ 30% → +1 分
- ✅ 用字多样性 ≥ 40% → +1 分

### 翻译准确性
- ≥ 0.8 相似度 → 3.0 分
- ≥ 0.6 相似度 → 2.5 分
- ≥ 0.4 相似度 → 2.0 分
- ≥ 0.2 相似度 → 1.0 分
- < 0.2 相似度 → 0.5 分

## 💡 使用技巧

### 提高识别准确度
1. **手动修正**：如果自动识别不准确，可在文本框中手动编辑
2. **重新录音**：点击"清除"按钮清空当前录音，重新开始
3. **环境优化**：避免在嘈杂环境中使用，关闭背景音乐

### 优化评分
1. **完整表达**：尽量完整翻译原文含义，避免遗漏关键信息
2. **标点规范**：用标点符号分隔句子，表示表达的完整性
3. **多样用词**：避免过度重复同一词汇或表述

### 学习记录
- 所有练习记录自动保存到浏览器本地存储
- 点击"学习记录"标签页查看历史记录和统计数据
- 支持 CSV 导出功能，便于离线分析

## 📞 常见问题

### Q1: 切换翻译方向后如何重新选择材料？
A: 点击"选择练习难度"的下拉菜单，重新选择一个难度即可自动加载新的材料。

### Q2: 英文识别不准确怎么办？
A: 
- 确保在安静环境中使用
- 发音要清晰，语速适中（不要太快）
- 如果自动识别结果不满意，可在文本框中手动修正

### Q3: 中→英模式下，参考翻译如何显示？
A: 点击"提交评分"后，参考框会显示对应的英文翻译。

### Q4: 如何删除某条学习记录？
A: 进入"学习记录"标签页，使用"清除所有记录"按钮。暂不支持单条删除，建议使用 CSV 备份后手动管理。

### Q5: 能否离线使用？
A: 翻译机需要网络连接（调用在线翻译 API），但实战演练模块的语音识别和评分在本地完成，可基本离线使用（仅限录音和识别功能）。
